# gabemdev.com

This is my website. 

## Running locally
Clone 
        $ git clone https://github.com/gabemdev/gabemdev.com.git
        $ cd gabemdev.com
    
Now you can start the server with [Shotgun](https://github.com/rtomayko/shotgun):
         $ bundle exec shotgun
Open <http://localhost:9393> in your browser
